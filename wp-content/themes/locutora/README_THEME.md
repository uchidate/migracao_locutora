# Tema Locutora

Este tema foi criado para reproduzir o visual de locutora.com em WordPress.

## Estrutura

- `style.css` - estilos do tema
- `functions.php` - registros de menu e enfileiramento de estilo
- `header.php` - cabeçalho com menu principal
- `footer.php` - rodapé simples
- `front-page.php` - homepage estática com layout principal
- `page.php` - template para páginas simples
- `page-sobre-nos.php` - template para a página Sobre Nós
- `page-servicos.php` - template para a página Áudios
- `page-contato.php` - template para a página Contato

## Ativação

1. Suba o tema para `wp-content/themes/locutora`
2. No painel WordPress, ative o tema Locutora
3. Configure o menu em Aparência > Menus
4. Crie páginas e selecione os templates personalizados
5. Em Configurações > Leitura, defina a página inicial como estática

## Compatibilidade Hostinger Compartilhado

- WordPress 6.x
- PHP 8.0 / 8.1 / 8.2
- MySQL / MariaDB
- Compatível com FTP / SFTP / Gerenciador de Arquivos
